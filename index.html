Hangman

HTML5 Arcade game by Ravichandran JV

View on GitHub Download .zip Download .tar.gz

Hangman - the arcade game for excelling in spelling and vocabulary.

Gamers - Download either of the archive files (zip or tar) and extract the four files. If you wish to increase the word count, you could simply add more words to the Javascript array.

Author

This game was created in 2011 by @ravichandranjv as a simple example to try out the new features of HTML5 Canvas.
